# deployment/ansible/roles/setting/sample.txt
본 Role은 초기 서버 구축 및 필요한 공통 OS 패키지와
서버 역할 별 Python 가상환경(venv) 및 pip 패키지 설치까지 수행된다

각 서버는 분기 처리되어 하나의 role 내에서 서버별 환경 구성을 한다.

- 서버별 필요 패키지

[common] 공통 os 패키지
  - vim, git, curl, wget, net-tools, unzip
  - firewalld, python3, python3-pip, python3-devel, gcc
: 기본 OS 환경 및 Python 실행 기반 구축

[ccmall_con] : Terraform/ansible 관리 서버
  - Terraform 1.5.7
  - ansible 2.15.0
  - awscli
  - openssh-clients
: 인프라 및 배포 제어 전용 서버 (서비스 실행 x)

[ccmall_web] : FastAPI 앱 실행 서버
  - Nginx 1.25
  - 디렉토리 생성 : /opt/ccmall-web
  - venv 생성: /opt/ccmall-web/venv

  (pip 패키지)
  - fastapi 0.109.0
  - jinja2 3.1.3
  - uvicorn 0.27.0
  - sqlalchemy 2.0.25
  - psycopg2-binary 2.9.9
  - python-dotenv 1.0.0
  - passlib 1.7.4
  - python-multipart 0.0.6
: 웹 어플리케이션 실행 서버

[ccmall_rec] : 복구 및 백업 서버
  - awscli
  - 디렉토리 생성: /opt/backup
  - venv 생성: /opt/backup/venv

  (pip 패키지)
  - apscheduler 3.10.4
  - boto3 1.34.0
  - python-dotenv 1.0.0
: 백업 스케줄, s3 업로드 처리, 데이터 아카이빙 수행

[onprem] : 로컬 운영 DB 서버
- 공통 패키지 설치
